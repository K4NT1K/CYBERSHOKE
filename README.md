# cybershoke

Репозиторий с расширением **IoHelper** для страниц системы модерации CYBERSHOKE.

## Содержимое

- `IoHelper/` — файлы расширения Chrome
- `IoHelper/privacy-policy.txt` — файл политики конфиденциальности

## Политика конфиденциальности

Доступ к доменам CYBERSHOKE необходим для работы расширения на страницах системы модерации. Расширение считывает информацию со страниц тикетов и отображает дополнительные инструменты для анализа нарушений и принятия решений модераторами.
